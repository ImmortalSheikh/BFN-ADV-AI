# Task 1 — Demand Prediction
### Bristol Regional Food Network — Advanced AI

---

## Overview

Analyses customer purchase history to predict frequently ordered items and provide personalised reorder suggestions. Produces demand forecasts for producers to help with stock planning.

---

## How to Run

### Step 1 — Install dependencies
```bash
pip install -r ../requirements.txt
```

### Step 2 — Generate dataset
```bash
python data.py
```

### Step 3 — Train the model
```bash
python train.py
```

### Step 4 — Evaluate the model
```bash
python evaluate.py
```

### Step 5 — Run predictions demo
```bash
python predict.py
```
